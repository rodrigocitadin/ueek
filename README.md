# UEEK Connect

para rodar o projeto, ira precisar do Node, e NPM

primeiro, clone o repo

```bash
git clone https://github.com/rodrigocitadin/ueek
```

```bash
cd ueek
```

Instale as dependencias

```bash
npm install
```

Rode o projeto

```
npm run dev
```

## Como usar

basta entrar no link `localhost:3000` no seu browser
